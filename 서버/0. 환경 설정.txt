비쥬얼 스크립트 -> C# Windows 콘솔 앱 -> 프로젝트 이름 Server 생성 -> 솔루션 안에서 새프로젝트 생성 
-> ServerCore 생성 -> DummyClient 생성 ->

아직 유니티에서 연결을 하지 않고 더미 클라이언트에서 확인을 하여 사용한다
경우에 따라서 클라이언트를 여러개 접속한것 처럼 패킷을 쏘는 역할도 함 테스트 서버임

DummyClient 하는 일
- 여러개 접속한 것 처럼 패킷을 쏘는 역할
- 테스팅 환경을 구축하는 위한 용도

ServerCore 하는 일
- 우리한테 필요한 핵심 기능



여러번 실행하는 방법
- 솔루션에서 속성 -> 여러프로젝트 실행 하면 디버그 콘솔창이 여러개 실행이 됨
